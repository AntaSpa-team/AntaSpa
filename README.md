\# Sistema Web Integral para la Optimización de la Gestión de Citas, CRM, Inventario y Catálogo Digital  

\## Anta Salón Spa y Barbería



\---



\## Propósito del Proyecto



El propósito de este proyecto es desarrollar un sistema web integral que permita optimizar y modernizar la gestión administrativa y operativa de Anta Salón Spa y Barbería, ubicado en la ciudad de Trujillo.



Actualmente, el negocio presenta diversas limitaciones en la organización de sus procesos internos debido al uso de herramientas tecnológicas desactualizadas y métodos manuales para la gestión de citas, clientes e inventario. Estas deficiencias generan conflictos en la programación de servicios, desorden en la atención al cliente y dificultades en el control de productos disponibles.



Ante esta problemática, el proyecto tiene como finalidad diseñar e implementar una solución tecnológica que centralice los procesos principales del negocio en una sola plataforma digital. Este sistema permitirá mejorar la organización interna, reducir errores operativos, facilitar la coordinación entre especialistas y fortalecer la atención al cliente.



Asimismo, el sistema contribuirá al crecimiento del negocio mediante la implementación de herramientas que permitan registrar información detallada de clientes, controlar el inventario en tiempo real y administrar un catálogo digital de productos. De esta manera, se busca mejorar la eficiencia operativa, optimizar el uso de los recursos y contribuir al incremento de la rentabilidad del negocio.



\---



\## Objetivo General



Desarrollar un sistema web integral que optimice la gestión administrativa, el control de citas, el catálogo digital y la fidelización de clientes en Anta Salón Spa y Barbería, con el fin de mejorar la organización interna y la eficiencia operativa del negocio.



\---



\## Objetivos Específicos



\- Implementar un módulo de gestión de citas con agenda interactiva que permita registrar y organizar la disponibilidad de los especialistas.

\- Desarrollar un sistema básico de gestión de clientes (CRM) que permita registrar información completa y almacenar el historial de servicios.

\- Diseñar un módulo de catálogo digital y control de inventario que permita gestionar productos y controlar el stock en tiempo real.

\- Implementar un sistema de autenticación y control de acceso basado en roles que garantice la seguridad de la información.



\---



\## Alcance del Proyecto



El proyecto comprende el desarrollo e implementación de un sistema web orientado a la gestión interna de Anta Salón Spa y Barbería. El sistema permitirá registrar citas, administrar información de clientes, controlar inventario y gestionar productos mediante un catálogo digital.



El sistema estará diseñado para mejorar la organización del trabajo, reducir errores en la programación de servicios y facilitar la toma de decisiones mediante el uso de información confiable.



No se contempla en esta etapa la implementación de reservas en línea por parte de los clientes, la integración con sistemas de pago electrónico ni el desarrollo de aplicaciones móviles.
